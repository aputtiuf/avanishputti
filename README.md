# Avanish Putti Ocean Liquid Glass Contact Card

Upload all files to a GitHub repository, then enable GitHub Pages.

## Included
- `index.html`
- `styles.css`
- `script.js`
- `avanish.jpg`
- `uf-logo.png`

## Background
The moving ocean shoreline is built with CSS animation, so it works on GitHub Pages without needing a large video file.

## NFC Setup
Once GitHub Pages is published, copy the live website URL and program your NFC tag to open that URL.
