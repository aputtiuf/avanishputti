# Avanish Putti Beach Liquid Glass Contact Card

Upload all files to a GitHub repository, then enable GitHub Pages.

## Included
- `index.html`
- `styles.css`
- `script.js`
- `avanish.jpg`
- `beach.jpg`
- `uf-logo.png`

## Background
Uses the uploaded aerial beach image with a slow premium Ken Burns-style drift animation.

## NFC Setup
Once GitHub Pages is published, copy the live website URL and program your NFC tag to open that URL.
